#Snake Game

A new implementation of the common snake game that has all the features of the the original game, but here there is a maths learning premise

![Css]

##Features
Leaderboard is created so players can challenge each other
Score is saved in each play
Every time tyhe snake eats an apple a maths question is asked chosen at random

##Future improvements
Change the landscape of the game
Create harder questions as the game goes on
